# `download.ASCII.file` downloads grid data.

This is an internal function. It downloads an ARCMAP ASCII grid file
from a URL.

## Usage

``` r
download.ASCII.file(
  url.string,
  ivar.url.ext,
  ivar.file.ext,
  ivar.timestep,
  data.type.label,
  workingFolder,
  datestring
)
```

## Value

A list variable of the file name and succes/failure flag.
