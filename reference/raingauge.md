# Example rain gauge data.

Daily data for rain gauge 63005 from 2010. It used by a vignette to
evaluate the extracted point data. The data is from Bathurst, NSW.

## Usage

``` r
data("raingauge")
```

## Format

data.frame of 8 columns and 365 rows.

## Source

Bureau of Meteorology, Australia. See
<https://www.bom.gov.au/climate/data/index.shtml>
