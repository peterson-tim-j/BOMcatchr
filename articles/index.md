# Articles

### All vignettes

- [Make source data
  grids](https://peterson-tim-j.github.io/AWAPer/articles/A_Make_data_grids.md):

  This vignette shows how to build the netCDF data file and then update
  it to new start and end dates.

- [Extract daily point
  precipitation](https://peterson-tim-j.github.io/AWAPer/articles/B_Point_rainfall.md):

  This vignette shows how to build a netCDF AWAP data file and then
  extract the point precipitation data at a few locations over a usr
  defined period. The extracted data is then compared with precipitation
  gauge data.

- [Extract daily area weighted potential PET and precipitation and
  map](https://peterson-tim-j.github.io/AWAPer/articles/C_Catchment_avg_ET_rainfall.md):

  This vignette shows how to build the netCDF data file and then extract
  the area weighted PET and precipitation data for a catchment over a
  user defined period. The errors introduced from not using area
  weighted methods (e.g centrod) are then demonstrated. output:
  rmarkdown::html_vignette

- [Extract various measures of
  evapotranspiration](https://peterson-tim-j.github.io/AWAPer/articles/D_Catchment_avg_ET_types.md):

  This vignette shows how to estimate various measures of the area
  weighted evapotranspiration.
