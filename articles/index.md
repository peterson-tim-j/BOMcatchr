# Articles

### All vignettes

- [Make source data
  grids](https://peterson-tim-j.github.io/BOMcatchr/articles/A_Make_data_grids.md):

  This vignette shows how to build the netCDF data file and then update
  it to new start and end dates.

- [Getting
  started](https://peterson-tim-j.github.io/BOMcatchr/articles/AWAPer.md):

  This vignette outlines the key steps of building the netCDF data and
  then extracting data.

- [Extract daily and monthly point
  precipitation](https://peterson-tim-j.github.io/BOMcatchr/articles/B_Point_rainfall.md):

  This vignette shows how to build the netCDF data file and then extract
  the point daily and monthly precipitation data at a few locations over
  a user defined period. The extracted data is then compared with
  precipitation gauge data at a daily and monthly time step.

- [Getting
  started](https://peterson-tim-j.github.io/BOMcatchr/articles/BOMcatchr.md):

  This vignette outlines the key steps of building the netCDF data and
  then extracting data.

- [Extract daily area weighted PET and
  precipitation](https://peterson-tim-j.github.io/BOMcatchr/articles/C_Catchment_avg_ET_rainfall.md):

  This vignette shows how to build the netCDF data file and then extract
  the area weighted PET and precipitation data for a catchment over a
  user defined period. The errors introduced from not using area
  weighted methods (e.g centroid) are then demonstrated.

- [Extract various measures of
  evapotranspiration](https://peterson-tim-j.github.io/BOMcatchr/articles/D_Catchment_avg_ET_types.md):

  This vignette shows how to estimate various measures of the area
  weighted evapotranspiration.

- [Extract water year area weighted
  precipitation](https://peterson-tim-j.github.io/BOMcatchr/articles/E_Water_year_catchment_rainfall.md):

  This vignette shows how to build the netCDF data file and then extract
  user defined water year area weighted precipitation data for a
  catchment.
